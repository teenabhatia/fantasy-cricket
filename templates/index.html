T20 World Cup

Manage Your Fantasy Cricket Team

This website allows you to manage your fantasy cricket team for the T20 World Cup.

Fields and Buttons:
  • Player Name: Enter the name of the player you want to add to a friend's team.
  • Friend Name: Enter the name of the friend who owns the player.
  • Add Player to Friend: Click this button to assign the player to the friend.
  • Short Name: Enter the short name or nickname of a player.
  • Full Name: Enter the full name of the player corresponding to the short name.
  • Add Name Mapping: Click this button to add a mapping from the short name to the full name.
  • Cricbuzz URL: Enter the URL of the match scorecard from Cricbuzz to update points.
  • Update Points: Click this button to update the points based on the match data from Cricbuzz.
  • Show Scores: Click this button to display the current scores of all players and friends.

Current Point System:
  • +1 point for each run scored
  • +25 points for each wicket taken
  • +5 points for each catch taken
  • +5 points for each runout
